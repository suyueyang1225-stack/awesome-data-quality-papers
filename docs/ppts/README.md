## Upload and Preview Workflow

For each paper:

1. Use the paper anchor shown in the table.
2. Upload the editable slide deck to `docs/ppts/files/{paper-anchor}.pptx`.
3. Export the deck as PDF and upload it to `docs/ppts/previews/{paper-anchor}.pdf`.
4. Update the row from `TBD` to links for both the source deck and preview.

GitHub can store `.pptx` files, but PDF is more reliable for browser preview. Keep the `.pptx` for editing and the `.pdf` for quick review.
